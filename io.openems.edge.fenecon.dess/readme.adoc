= FENECON DESS

Applies to multiple similar products like the FENECON by BYD PRO Hybrid.

Implemented Natures::
- SymmetricEss
- AsymmetricEss
- EssDcCharger
- AsymmetricMeter (for Grid and AC-connected PV)
- SymmetricMeter (for Grid and AC-connected PV)
